### Breakpoints

`xs < 576 <= sm < 768 <= md < 992 <= lg < 1200 <= xl` [In pixels] <br />
If sm is specified then that class will work at >= sm devices. <br />
Example: <br />
```
    <p class="text-sm-center">This is some text that is centered on >= sm devices</p>

```
